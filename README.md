# delivery-robot

一个用于系统学习 Rust 和 ROS2 的实践项目。

## 项目目标

通过构建一个室内自主配送机器人系统，系统掌握：

- Rust 语言核心语法与工程实践
- ROS2 核心通信模型与机器人软件架构
- 路径规划、任务调度、状态管理、仿真验证
- 从单模块代码到多进程机器人系统的完整开发流程

## 目录结构

- `rust_ws`：Rust 核心工作区，放算法、状态机、公共库、实验代码
- `ros2_ws`：ROS2 工作区，放节点、消息、启动文件、仿真相关内容
- `docs`：架构设计、学习笔记、阶段任务拆解
- `scripts`：辅助脚本

## 学习原则

1. 不只追求能跑，要理解每个模块为什么这样设计。
2. 每完成一个阶段，就补齐文档、测试和一次 git 提交。
3. 先做最小闭环，再逐步增加复杂度。
